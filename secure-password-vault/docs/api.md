# Secure Password Vault Platform – API Documentation

## Endpoints

### /analyze-password
- POST
- Analyze password strength, entropy, patterns

### /check-breach
- POST
- Check if password/email appears in breach datasets

### /security-score
- GET
- Get overall security score and analytics

### /login-alert
- POST
- Triggered on suspicious login events

### /generate-password
- POST
- Generate strong passwords or passphrases

### /simulate-attack
- POST
- Simulate dictionary/brute-force/credential stuffing attacks

### /backup-vault
- POST
- Create encrypted backup of vault data

---

All endpoints require local access. No paid APIs required.
