# Vaultwarden Admin Configuration

# To enable the admin panel, set the ADMIN_TOKEN environment variable in docker-compose.yml.
# Example:
#   environment:
#     - ADMIN_TOKEN=changeme_admin_token
#
# Access the admin panel at: http://localhost:8080/admin
#
# Admin panel allows:
# - User management
# - Organization management
# - Security settings
#
# For more options, see: https://github.com/dani-garcia/vaultwarden/wiki/Enabling-admin-page
